## Dove trovare le immagini:

### Fonti consigliate (uso gratuito con attribuzione):

1. **Unsplash** (https://unsplash.com/)
   - Cerca: "Abruzzo mountains", "Italian Alps winter", "mountain village Italy"

2. **Pexels** (https://www.pexels.com/)
   - Cerca: "ski resort Italy", "Italian mountains", "alpine village"

3. **Pixabay** (https://pixabay.com/)
   - Cerca: "Maiella", "Abruzzo", "montagna Italia"

4. **Foto proprie**
   - Scatta foto durante una visita a Campo di Giove
   - Usa le webcam locali per screenshot (con permesso)

### Webcam Campo di Giove (per screenshot panoramici):
- Skiinfo: https://www.skiinfo.it/abruzzo/campo-di-giove-maiella/webcam
- Dove Sciare: https://www.dovesciare.it/webcam/campo_giove

## Specifiche tecniche:

- **Formato**: JPG o WebP
- **Risoluzione**: 1920x1080 px (Full HD orizzontale)
- **Proporzioni**: 16:9 (orizzontale)
- **Dimensione file**: Max 500 KB (ottimizzate per web)
- **Orientamento**: Orizzontale (landscape)
- **Qualità**: Alta, ma compressa per caricamento veloce

## Ottimizzazione immagini:

Prima di caricare, ottimizza le immagini con:
- [TinyJPG](https://tinyjpg.com/) - Compressione JPG
- [Squoosh](https://squoosh.app/) - Editor e compressore
- [ImageOptim](https://imageoptim.com/) - Per Mac

## Note:

- Le immagini devono evocare la bellezza di Campo di Giove
- Preferire foto luminose e di alta qualità
- Evitare immagini con watermark o loghi
- Se usi foto da internet, verifica sempre i diritti d'uso
- Le foto dovrebbero rappresentare le 4 stagioni o diverse attrazioni
